# A basic python clock
A clock coded in python based around pygame

Features:
Weather
Time
Date

**Libraries required:**

Pygame:
```console
$ python3 -m pip install -U pygame --user
```

time:
Should already be installed with python

requests:
```console
$ python -m pip install requests
```

json:
```console
$ python -m pip install json
```

io:
Should already be installed with python
